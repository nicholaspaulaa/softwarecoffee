# BrewMaster POS ☕

**Sistema de Ponto de Venda focado em produtividade e experiência do cliente para a cafeteria Coffee House.**

Este projeto é um MVP (Minimum Viable Product) desenvolvido como trabalho acadêmico de Engenharia de Software. O sistema visa eliminar comandas de papel ilegíveis, agilizar o fechamento de caixa e gerenciar o programa de pontos de fidelidade.

## Funcionalidades do MVP
- **CRUD de Clientes:** Cadastro e consulta automatizada para o programa de pontos (R$ 1,00 = 1 ponto).
- **CRUD de Produtos:** Gerenciamento do cardápio básico de cafés e confeitaria artesanal.
- **Fluxo de Vendas:** Abertura, adição de itens e fechamento de pedidos com acúmulo de pontos automático.

## Tecnologias Utilizadas
- **Linguagem:** Java (Orientação a Objetos)
- **Interface Gráfica:** Java Swing (VIEW)
- **Testes Automatizados:** JUnit (Testes de Unidade)
- **Arquitetura:** Padrão MVC (Model-View-Controller) integrado à camada DAO (Data Access Object).
